# Moon Cafe Chatbot

A small browser-based AI-style chatbot with a cozy Moon Cafe theme. It runs fully in the browser and uses a simple response engine to answer like a friendly barista on a lunar station.

## Files

- `index.html` - chatbot page
- `style.css` - interface styling
- `script.js` - chat behavior and themed responses

## Run

Open `index.html` in a browser.

## Try asking

- `hello`
- `menu`
- `recommend a drink`
- `space coffee`
- `help me relax`
