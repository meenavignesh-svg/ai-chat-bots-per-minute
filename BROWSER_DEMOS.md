# Browser Demos

Previously created chatbots now include browser demo behavior.

For static chatbots, open the chatbot's `index.html` file to preview the conversation flow. For AI-backed chatbots, the browser UI now falls back to themed demo replies when the backend, deployment, or API key is not available.

## Demo-enabled AI-backed chatbots

- Lab Mentor Chatbot
- Ocean Medic Chatbot
- Forest Ranger Chatbot
- City Chef Chatbot
- Museum Guide Chatbot
- Fitness Coach Chatbot
- Travel Planner Chatbot
- Study Buddy Chatbot
- Budget Mentor Chatbot

## Real AI mode

Some projects also include a backend or Vercel serverless route for real AI responses. Real AI mode requires a proper deployment and, depending on the project, either a server-side `OPENAI_API_KEY` or a visitor-provided OpenAI API key.

The browser demo is for previewing the chatbot experience without setup.
